============================================================
MAINTAINING THIS LIBRARY
============================================================

When adding new assignments:
- Follow the entry format exactly -- every field must be completed
- Tag therapeutic goals from the taxonomy in Section 1, ranked by relevance with primary goal first
- Write the Therapeutic Framing field with care -- this is the SCBF identity of the assignment
- Update the Assignment Index in Section 2
- Update the Goal Taxonomy to remove the [GAP] flag if the new assignment addresses a gap goal

When updating existing assignments:
- Update the Therapeutic Framing if SCBF methodology guidance changes
- Update grade ranges if field experience shows the assignment works better or worse at certain ages
- Do not change Student-Facing Components or Instructions without also reviewing the Therapeutic Goals
