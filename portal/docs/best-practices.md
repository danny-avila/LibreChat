# Bonnes Pratiques et Support

## Rédiger des requêtes efficaces

Pour obtenir les meilleures réponses possibles :
*   **Soyez précis** : Formulez clairement votre question ou demande.
*   **Fournissez du contexte** : Donnez les informations pertinentes pour aider le modèle.
*   **Structurez vos demandes** : Utilisez des listes ou des étapes si nécessaire.
*   **Itérez** : N'hésitez pas à reformuler ou préciser votre demande.

## Expérimenter avec les modèles

Chaque modèle d'IA a ses propres forces :
*   Essayez différents modèles pour la même tâche.
*   Comparez les réponses pour identifier les approches les plus utiles.
*   Notez quel modèle fonctionne le mieux pour vos types de requêtes.

Cette expérimentation vous aidera à développer une intuition sur quand utiliser chaque modèle.

## Support et ressources

Si vous rencontrez des difficultés ou avez des questions, vous pouvez contacter le support à l'adresse suivante : **[accesllm@etsmtl.ca](mailto:accesllm@etsmtl.ca)**.

### Aller plus loin

Pour approfondir votre utilisation de l'IA générative dans un contexte académique :
*   Explorez les guides de cas d'usage spécifiques à votre domaine.
*   Partagez vos meilleures pratiques avec vos collègues.
*   Suivez les formations complémentaires sur l'utilisation éthique de l'IA.
