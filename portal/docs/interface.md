# Navigation et Interface


## L'interface principale
![Interface principale](/img/tutorial/main_interface.png)

Une fois connecté, vous verrez l'interface principale qui se compose de trois zones principales :

1.  **Panneau latéral gauche** : Historique de vos conversations.
2.  **Zone centrale** : Espace de conversation avec le modèle d'IA.
3.  **Barre supérieure** : Sélection du modèle et paramètres.

## Sélectionner un modèle

Pour choisir le modèle d'IA avec lequel vous souhaitez interagir :

1.  Cliquez sur le menu déroulant des modèles dans la barre supérieure.
2.  Parcourez les modèles disponibles (OpenAI GPT, Claude, etc.).
3.  Sélectionnez le modèle de votre choix.

Chaque modèle possède des caractéristiques uniques. N'hésitez pas à en essayer plusieurs pour identifier celui qui répond le mieux à vos besoins.

## Débuter une conversation

1.  Cliquez sur le bouton **Nouvelle conversation**.
2.  Tapez votre requête dans le champ de saisie au centre de l'écran.
3.  Appuyez sur **Entrée** ou cliquez sur le bouton d'envoi.

Le modèle générera une réponse en quelques secondes.

## Comparaison des modèles
![Deuxième modèle](/img/tutorial/second_model.png)

LibreChat vous permet d’envoyer une requête simultanément à deux modèles, ce qui vous permet de comparer les résultats en parallèle.

Pour utiliser cette fonction :
1.  Choisissez un premier modèle, comme pour une conversation normale.
2.  Cliquez sur le **bouton plus (+)** à droite du menu de sélections des modèles.
3.  Choisissez le deuxième modèle que vous désirez.
