# Gestion des conversations

Toutes vos conversations sont automatiquement sauvegardées et accessibles dans le panneau latéral gauche. Vous pouvez :

## Renommer une conversation

Pour faciliter le repérage de vos conversations :
1.  Survolez la conversation dans le panneau latéral.
2.  Cliquez sur l'icône de modification (crayon ou trois points).
3.  Choisissez **Renommer**.
4.  Entrez un titre descriptif et sauvegardez.

## Supprimer une conversation

Si vous souhaitez supprimer une conversation :
1.  Localisez la conversation dans le panneau latéral.
2.  Cliquez sur l'icône de menu (trois points).
3.  Sélectionnez **Supprimer**.

:::danger Attention
La suppression est définitive et ne peut être annulée.
:::

Si vous souhaitez ajouter une conversation aux signets  :
1.  Sélectionner une conversation.
2.  Cliquer sur le bouton **Ajouter des signets**.
3.  Choisir un signet existant ou en créer un nouveau.
4.  Gérer vos signets (modifier, supprimer) dans la barre latérale.
5.  Filtrer vos conversations par signet pour regrouper facilement vos échanges.
