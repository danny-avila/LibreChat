# Analyse de Documents

**AccèsLLM** vous permet de téléverser divers types de documents pour analyse, résumé ou discussion.

## Types de fichiers supportés

Les formats supportés incluent :
*   **Documents texte** : PDF, DOCX, TXT
*   **Images** : PNG, JPG, JPEG
*   **Feuilles de calcul** : XLSX, CSV

## Procédure de téléversement

Pour ajouter un document à votre conversation :
1.  Cliquez sur l'icône de trombone ou de téléversement dans le champ de saisie.
2.  Sélectionnez le fichier depuis votre ordinateur.
3.  Patientez pendant le téléversement (une barre de progression apparaîtra).
4.  Une fois le document chargé, posez votre question concernant le contenu.

## Exemples d'utilisation

Voici quelques façons d'exploiter cette fonctionnalité :

*   **Résumer un article scientifique** : Téléversez l'article et demandez un résumé des points principaux.
*   **Analyser des données** : Partagez un fichier CSV et posez des questions sur les tendances.
*   **Extraire des informations** : Téléversez un rapport et demandez des détails spécifiques.
*   **Comparer des documents** : Téléversez plusieurs fichiers pour une analyse comparative.
